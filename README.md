Play space invaders on your computer and or with Arduino Uno

SpaceInvader.py runs completely on computer with no arduino code or extension needed 
Run SpaceInvader.py 
Use the left and right arrow keys to control the moement of the player. 
Use space to fire bullets onto the enmeies.

SpaceInvaders.py runs with arduino

Materials Needed: Arduino Uno Potentiometer Button Wires

Arduino process: Connect the left pin of the potentiometer to gnd, the right pin to 5V and the middle pin to A0. 
Connect the left leg of the button to the same gnd and the right leg to 7. 
Run the Arduino Code (Open the serial monitor to check if it works). 
Close the serial monitor. 
Run SpaceInvaders.py 
Turn the dial of the potentiometer to control the movement of the player (left and right). 
Press the button to fire a bullet onto the enemies.
